# Tutorial 3 - Game Development
### Nayla Farah Nida [2306213426]

Level 1 (```Level1.tscn```) merupakan hasil dari mengerjakan tutorial. Dalam latihan mandiri, saya membuat level baru (```Level2.tscn```) dengan beberapa tantangan seperti jurang, moving platform, objek ```Mouse.tscn``` sebagai enemy, 
dan ```Star.tscn``` yang berjatuhan dari langit dengan memanfaatkan ```spawner```. Jika pengguna mengenai ```Mouse``` atau ```Star``` pemain akan kembali ke posisi awal (restart).
Objektif pada level ini adalah mencapai 'Rektorat' di mana pemain harus melewati jurang, ```Mouse```, serta falling ```Star```.
